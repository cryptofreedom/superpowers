# jason-superpowers (personal fork)

Personal Claude Code–only fork of [obra/superpowers](https://github.com/obra/superpowers), customized for **Opus 4.7**.

Multi-harness support (Codex / Cursor / OpenCode / Gemini / Copilot) and seven core skills (`executing-plans`, `writing-plans`, `using-git-worktrees`, `dispatching-parallel-agents`, `subagent-driven-development`, `requesting-code-review`, `finishing-a-development-branch`) have been removed because Claude Code covers them natively. The `test-driven-development` skill has been replaced with a dual-loop Java/Spring–flavored framework.

See `archive/README.md` for the full removal log and recovery references.

This repo is not a contribution target — file issues with upstream instead.
